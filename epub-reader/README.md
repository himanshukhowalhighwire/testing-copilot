# EPUB Reader

A minimal web application to read EPUB files. Upload any EPUB file, view its content, and switch between pages.

## Features
- Upload and open any EPUB file
- Navigate between pages (spine items)

## Getting Started

1. Install dependencies:
   ```bash
   cd epub-reader
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage
- Click the file input to upload an `.epub` file.
- Use the Previous/Next buttons to navigate pages.

---

Built with React and [epub.js](https://github.com/futurepress/epub.js/).
